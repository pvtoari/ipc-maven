## Ejecución

### Usando wrapper de Maven

Para ejecutar el proyecto usando el wrapper de Maven, se debe ejecutar el siguiente comando:

**En Unix o similares:**
```bash
./mvnw clean install javafx:run
```

**En Windows:**
```bash
mvnw.cmd clean install javafx:run
```

### Usando Maven

Para ejecutar el proyecto usando Maven, se debe ejecutar el siguiente comando:

```bash
mvn clean install javafx:run
```

*Para ambos casos, se debería abrir una ventana con un menú principal, en el cual se puede seleccionar la práctica a ejecutar.*
